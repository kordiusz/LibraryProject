A sophisticated **Library Management System** designed for customers to handle the full lifecycle of book borrowing. This application provides a streamlined interface for managing personal collections, checking real-time availability, and configuring automated notifications for high-demand titles.

* **Technology Stack:** JavaFX, FXML, SQLite.
* **Architecture:** MVC (Model-View-Controller) for clean separation of concerns.
* **Key Feature:** Intelligent notification system for "missing pieces" (out-of-stock items).
